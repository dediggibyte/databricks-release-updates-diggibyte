Beta

This feature is in [Beta](/aws/en/release-notes/release-types). Workspace admins can control access to this feature from the **Previews** page. See [Manage Databricks previews](/aws/en/admin/workspace-settings/manage-previews).

The managed Square connector in Lakeflow Connect ingests point-of-sale data such as payments, orders, customers, catalog items, and inventory from Square into Databricks.

## Feature availability[​](#feature-availability "Direct link to feature-availability")

| Feature | Availability |
| --- | --- |
| UI-based pipeline authoring | Supported |
| API-based pipeline authoring | Supported |
| Declarative Automation Bundles | Supported |
| Incremental ingestion | Supported |
| Unity Catalog governance | Supported |
| Orchestration using Databricks Workflows | Supported |
| API-based column selection and deselection | Supported |
| API-based row filtering | Not supported |
| SCD Type 2 | Supported |
| Automated schema evolution: New and deleted columns | Supported |
| Automated schema evolution: Data type changes | Not supported |
| Automated schema evolution: Column renames | Not supported  Requires a full refresh. |

## Authentication methods[​](#authentication-methods "Direct link to authentication-methods")

| Authentication method | Availability |
| --- | --- |
| OAuth U2M | Not supported |
| OAuth M2M | Not supported |
| Basic authentication (username/password) | Not supported |
| Basic authentication (API key) | Supported  Square API token. |

## Ingest from Square in 3 steps[​](#ingest-from-square-in-3-steps "Direct link to ingest-from-square-in-3-steps")

note

Before starting, review the [Databricks user persona](/aws/en/connect/managed-ingestion), [supported interfaces](/aws/en/ingestion/lakeflow-connect/faq#interfaces), [ingestion frequency](/aws/en/ldp/pipeline-mode), and [common patterns](/aws/en/ingestion/lakeflow-connect/common-patterns).

## Start ingesting from Square[​](#start-ingesting-from-square "Direct link to Start ingesting from Square")

1. **[Configure Square for ingestion](/aws/en/ingestion/lakeflow-connect/square-source-setup)** (Admins) — Set up Square to authenticate with Databricks.
2. **[Create a Unity Catalog connection](/aws/en/ingestion/lakeflow-connect/square-connection)** (Admins) — Create a connection in Catalog Explorer so non-admins can create pipelines.
3. **[Create an ingestion pipeline](/aws/en/ingestion/lakeflow-connect/square-pipeline)** (Admins or non-admins) — Select any supported interface and create a pipeline from an existing connection.
