This page includes a schema reference of the Genie Code events system table and an example that uses the data in a dashboard. Each row in this system table represents a message sent by a user in either Genie Code window or the in-cell Assistant. For more information about the Genie Code, see [Genie Code](/aws/en/genie-code/).

**Table path**: This system table is located at `system.access.assistant_events`.

note

As of May 7, 2026, this system table records only user-submitted Genie Code interactions, such as chat threads, agent mode, and the inline assistant. Background and automated requests are no longer logged as events. These include autocomplete (ghost text), quick-fix classifiers, automatic renaming, agent-search indexing, and evaluation and safety checks.

Because these requests were logged before May 7, 2026, daily event counts from earlier dates can appear higher than actual user activity. A decrease in counts around this date is expected and does not indicate a drop in usage.

## Assistant events schema[​](#assistant-events-schema "Direct link to Assistant events schema")

The Genie Code events system table uses the following schema:

| Event | Data type | Description | Example |
| --- | --- | --- | --- |
| `event_id` | string | A unique ID for this event. | `80899c5dfe4b6b9505f3ea5a90e9c97f7ab9fa8f` |
| `account_id` | string | ID of the account. | `ca886134-876c-4671-a38b-332edf48c602` |
| `workspace_id` | string | ID of the workspace. | `2548836972759138` |
| `event_time` | timestamp | Time that the event happened. Timezone information is recorded at the end of the value with `+00:00` representing UTC. | `2024-01-05T00:00:00.000+00:00` |
| `event_date` | string | Date that the event happened. | `2024-01-05` |
| `user_agent` | string | Origination of request. | `Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/121.0.0.0 Safari/537.36` |
| `initiated_by` | string | Email of the user initiating the request. | `abc@databricks.com` |

## Genie Code sample dashboard[​](#genie-code-sample-dashboard "Direct link to genie-code-sample-dashboard")

Use system table data to monitor the adoption of Genie Code. The dashboard shows the following:

* Active users by day and month
* Active users per workspace
* Top users overall
* Submission data: per workspace and total

To visualize the system table activity for Genie Code, download the [Genie Code sample dashboard file](https://github.com/databricks-demos/dbdemos-notebooks/blob/main/product_demos/Unity-Catalog/uc-04-system-tables/_resources/dashboards/genie-code-metrics) from GitHub.

After you download the JSON file, import the dashboard into your workspace. For instructions on importing dashboards, see [Import a dashboard file](/aws/en/dashboards/automate/import-export#import).
